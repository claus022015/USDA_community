Project Plan¶

Ontomatica will deliver a prototype  label application within 60 days of contract start date.

Ontomatica will deliver complete system  within 120 days of finalizing a modified specification.

Ontomatica anticipates USDA Label Project will be open-for-business on or about May 26, 2015.

Release 0  is contract signing and start date.

Release 1  is Prototype. Prototype will reuse and modify existing Ontomatica capabilities. For example, deposition services and REST services will not be available for Release 1.

Release 2  is modified specification  jointly developed by USDA and Ontomatica.

Release 3  delivers complete system. Release 3 trains USDA level 1 support  on use, management and implementation of complete system.
